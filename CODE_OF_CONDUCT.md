# Code of Conduct

This repository exists to help people recover a useful Claude Code behavior
without myth-making.

That means our working style should be:

- respectful
- direct
- technically honest
- patient with new users

Behavior that helps:

- critique code and claims, not people
- explain why something is wrong
- help others reproduce issues without ridicule

Behavior that does not belong here:

- harassment or intimidation
- personal attacks
- sexualized or abusive language
- mocking new users for not knowing terminal details

This applies to issues, discussions, pull requests, and linked beta support
spaces.

Maintainers may remove comments, close threads, or block participation when
behavior harms the project or its contributors.
