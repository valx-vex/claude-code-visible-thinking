# Claude Code Visible Thinking `v0.1.0-beta.1`

Quiet beta packaging release.

## What This Is

This release turns the repo from a useful setup note into a supportable beta
product:

- version-aware verification
- support bundle
- governance and contribution docs
- issue and discussion templates
- CI for repo-surface checks

## What Works

- apply the canonical visible-thinking settings
- back up existing Claude config
- verify machine-readable `thinking` output on supported Claude Code versions
- guide operators toward transcript-visible reasoning when chat rendering varies

## Known Limits

- unsupported older Claude Code versions may not expose the documented behavior
- this is still an unofficial repository
- raw full chain-of-thought is not part of the promise
