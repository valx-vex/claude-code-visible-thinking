# Roadmap

## Current Phase

`v0.1.0-beta.1` is the quiet-beta packaging phase:

- reproducible settings path
- honest verification boundary
- support and contribution scaffolding
- release discipline before public marketing

## Near-Term

- make supported-version checks explicit
- improve support bundle quality
- normalize GitHub Issues and Discussions for outside testers
- keep validation receipts current when Claude Code stable changes

## Toward v1.0.0

- repeatable second-machine validation
- better operator onboarding for terminal-cautious users
- enough outside confirmations that the repo works beyond the maintainer machines

## Explicit Non-Goals Right Now

- promising full raw thought recovery
- broad operating system support without validation
- turning this repo into a general Claude Code modification framework
